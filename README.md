## 运行
程序已经打包好放在 [release](https://github.com/liuzhuoling2011/baidupcs-web/releases) 页面, 直接双击就可以使用
如果遇到程序启动错误, 应该是端口占用导致的, 可以指定端口运行

```./BaiduPCS-Go-xxx web --port 12345```
